README.md

Mini-project 2: HTML Head Checklist được xây dựng nhằm tìm hiểu và thực hành cấu hình phần Head trong HTML5. Nội dung bao gồm Meta Charset, Meta Viewport, Title, Link CSS, Script JavaScript, Comment HTML và quy tắc thụt dòng.

Mục tiêu dự án

  • Tìm hiểu Meta Charset UTF-8
  • Tìm hiểu Meta Viewport
  • Tìm hiểu Title
  • Tìm hiểu Link CSS
  • Tìm hiểu Script JavaScript
  • Tìm hiểu Comment HTML
  • Tìm hiểu quy tắc thụt dòng

Ví dụ cấu trúc Head chuẩn

<head>
    <meta charset="UTF-8">

    <meta name="viewport"
          content="width=device-width, initial-scale=1.0">

    <title>Mini-project 2</title>

    <link rel="stylesheet"
          href="css/style.css">

    <script src="js/script.js"></script>
</head>
